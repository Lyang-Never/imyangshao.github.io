简介

简单说一下，图床呢，也就是专门提供存储图片的地方，我们只要通过图床提供的 API 接口，把图片上传上去，就可以通过外链访问了，根本不用操心图片是怎么存的，硬盘空间不够了，硬盘坏了，访问速度比较慢等等问题。但是，也存在一个问题，图床挂了的话，就GG了。话不多说，开搞！(以下主要针对windows)

二、PicGo获取和配置

2.1PicGo的简介和下载

PicGo 算得上一款比较优秀的图床工具。它是一款用 Electron-vue 开发的软件，可以支持微博，七牛云，腾讯云COS，又拍云，GitHub，阿里云OSS，SM.MS，imgur 等8种常用图床，功能强大，简单易用。我们这里选择Github。

移步这里：下载地址

2.2PicGo的配置

仓库名："账户名/仓库名"

分支名：建议填写 "master"

Token：把刚才github生成的token复制过来即可

路径：就是相应的会在刚才你的repository下生成一个存储你图片的文件夹

自定义域名：按照以下格式填写，"https://raw.githubusercontent.com/用户名/RepositoryName/分支名"，就是最后生成图片的访问链接为"自定义域名+图片名称"


最后，点击确定即可。

In the end,可以自行于上传区体验。
